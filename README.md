## React Nepali Date Picker
Nepali date picker UI input field component for React 

## Usage


```
import { NepaliC } from "nepalidatepicker-react";

```

```
   <NepaliC
        value={date}
        readOnly={false}
        default={"१२-१२-१९९९"}
        format={"MM-DD-YYYY"}
        onChangeHandler={handleChange}
      />
```
